C# Variables

In C#, you can declare variables to store and manipulate data. 

// Example C# variables
string name = "Alice";
int age = 30;

C# Data Types

C# supports various data types, including strings, integers, floats, and more. 

// Example C# data types
string name = "Alice";
int age = 30;
float salary = 45000.50f;

C# Operators

C# provides operators for performing operations on variables and values. 

// Example C# operators
int x = 10;
int y = 5;
int sum = x + y;
int difference = x - y;
int product = x * y;

C# Conditional Statements

Use conditional statements like if-else in C# for decision-making in your programs. 

// Example C# if-else statement
int age = 18;
if (age >= 18)
{
    Console.WriteLine("You are an adult.");
}
else
{
    Console.WriteLine("You are a minor.");
}

C# Loops

C# supports various types of loops like for, while, and do-while for repetitive tasks. 

// Example C# for loop
for (int i = 1; i <= 5; i++)
{
    Console.WriteLine("Iteration " + i);
}

C# Functions

Functions in C# are defined using the `static` keyword and allow you to reuse code. 

// Example C# function
using System;

class Program
{
    static void Greet(string name)
    {
        Console.WriteLine("Hello, " + name + "!");
    }

    static void Main()
    {
        Greet("Bob");
    }
}
