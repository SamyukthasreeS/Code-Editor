Ruby Variables

In Ruby, you can declare variables to store and manipulate data. 

# Example Ruby variables
name = "Alice"
age = 30

Ruby Data Types

Ruby supports various data types, including strings, numbers, arrays, and more. 

# Example Ruby data types
string_variable = "Ruby"
integer_variable = 42
array_variable = [1, 2, 3, 4, 5]

Ruby Operators

Ruby provides operators for performing operations on variables and values. 

# Example Ruby operators
x = 10
y = 5
sum = x + y
difference = x - y
product = x * y

Ruby Conditional Statements

Use conditional statements like if-else in Ruby for decision-making in your programs. 

# Example Ruby if-else statement
age = 18
if age >= 18
    puts "You are an adult."
else
    puts "You are a minor."
end

Ruby Loops

Ruby supports various types of loops like for, while, and until for repetitive tasks. 

# Example Ruby for loop
for i in 1..5
    puts "Iteration #{i}"
end

Ruby Functions

Functions in Ruby are defined using the `def` keyword and allow you to reuse code. 

# Example Ruby function
def greet(name)
    puts "Hello, #{name}!"
end

# Call the function
greet("Bob")
