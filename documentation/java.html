Java provides control flow statements like if-else, switch, loops, and more to control the flow of a program.

Java If-else

Using if-else statements in Java to make decisions in your program. 

// Example Java if-else statement
int age = 18;
if (age >= 18) {
    System.out.println("You are an adult.");
} else {
    System.out.println("You are a minor.");
}

Java Switch

Implementing switch statements in Java to select one of many code blocks to be executed. 

// Example Java switch statement
int day = 3;
String dayName;
switch (day) {
    case 1:
        dayName = "Monday";
        break;
    case 2:
        dayName = "Tuesday";
        break;
    // Add cases for other days
    default:
        dayName = "Invalid day";
}

Java For Loop

Using for loops in Java for iterating over a range of values. 

// Example Java for loop
for (int i = 1; i <= 5; i++) {
    System.out.println("Iteration " + i);
}

Java While Loop

Implementing while loops in Java for repetitive tasks. 

// Example Java while loop
int count = 0;
while (count < 3) {
    System.out.println("Count: " + count);
    count++;
}

Java Do While Loop

Using do-while loops in Java to execute a block of code at least once, even if the condition is false. 

// Example Java do-while loop
int num = 0;
do {
    System.out.println("Number: " + num);
    num++;
} while (num < 3);

Java Break

Using the break statement in Java to exit a loop or switch statement prematurely. 

// Example Java break statement
for (int i = 1; i <= 5; i++) {
    if (i == 3) {
        break; // Exit the loop when i equals 3
    }
    System.out.println("Iteration " + i);
}

Java Continue

Using the continue statement in Java to skip the current iteration of a loop and continue with the next. 

// Example Java continue statement
for (int i = 1; i <= 5; i++) {
    if (i == 3) {
        continue; // Skip iteration when i equals 3
    }
    System.out.println("Iteration " + i);
}
