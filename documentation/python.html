Python Introduction

Python is a high-level programming language known for its simplicity and readability. 

# Example Python code
print("Hello, Python!")

Python keywords are reserved words used to define the language's syntax and structure. 

# Example Python keywords

Python Variables, Constants, and Literals

In Python, you can declare variables, constants, and use literals. 

# Example Python variables

name = "John"

# Example Python constants
PI = 3.14159

# Example Python literals
num_literal = 42

Python Data Types

Python supports various data types, including integers, floats, strings, and more. 

# Example Python data types
integer_variable = 42

string_variable = "Python"

Python Operators

Python supports various types of operators for different operations. 

# Example Python arithmetic operators
x = 10
y = 5
addition = x + y
subtraction = x - y
multiplication = x * y
division = x / y

# Example Python comparison operators
a = 10
b = 20
is_equal = a == b
is_not_equal = a != b
greater_than = a > b
less_than = a < b

# Example Python logical operators
is_true = True
is_false = False
logical_and = is_true and is_false
logical_or = is_true or is_false
logical_not = not is_true

Python Name, Namespace, & Scope

In Python, variables have different scopes: local, enclosing, and global. 

# Example Python variable scope
global_variable = 10

def my_function():
    local_variable = 20
    print(global_variable)  # Access global variable

# Access global_variable
my_function()

Python Flow of Control

Python provides if, for, while, and other statements for flow control. 

# Example Python if statement
x = 10
if x > 5:
    print("x is greater than 5")
else:
    print("x is not greater than 5")

# Example Python for loop
for i in range(5):
    print(i)

# Example Python while loop
count = 0
while count < 3:
    print(count)
    count += 1

Python Functions

Functions are reusable blocks of code in Python. 

# Example Python function
def greet(name):
    print("Hello, " + name + "!")

# Call the function
greet("Alice")
