C++ Basic

C++ is a powerful programming language with a rich set of features. This section covers the basics of C++ programming. 

// Example C++ program
#include

    std::cout << "Hello, C++!" << std::endl;

C++ Tutorial: What is C++?

C++ is a general-purpose programming language that extends the C programming language with features like classes and objects. It is widely used for system programming, game development, and more.

C++ Basic Syntax

C++ syntax defines how programs are written in the language. It includes rules for statements, declarations, and more. 

// Example C++ syntax
int main() {
    int x = 5; // Variable declaration and initialization
    std::cout << "The value of x is: " << x << std::endl;
    return 0;
}

C++ Variable Scope

In C++, variables can have different scopes, such as local and global. 

// Example C++ variable scope
#include 

int globalVariable = 10; // Global variable

int main() {
    int localVariable = 20; // Local variable
    std::cout << "Global variable: " << globalVariable << std::endl;
    std::cout << "Local variable: " << localVariable << std::endl;
    return 0;
}

C++ Constants/Literals

C++ allows you to define constants and use literals in your code. 

// Example C++ constants and literals
const double PI = 3.14159265359; // Constant
int literalInteger = 42; // Integer literal
double literalDouble = 3.14; // Double literal
char literalChar = 'A'; // Character literal

C++ Modifier Types

C++ provides modifiers like const and volatile to alter the properties of variables. 

// Example C++ modifier types
int const constantValue = 100; // Constant variable
volatile int volatileValue = 200; // Volatile variable

C++ Storage Classes

C++ has several storage classes like auto, static, and extern to define variable storage duration and scope. 

// Example C++ storage classes
auto int autoVariable = 42; // Auto storage class
static int staticVariable = 100; // Static storage class
extern int externVariable; // Extern storage class (defined in another file)

C++ Operators

C++ provides various operators for performing operations on variables and values. 

// Example C++ operators
int a = 10, b = 5;
int addition = a + b; // Addition operator
int subtraction = a - b; // Subtraction operator
int multiplication = a * b; // Multiplication operator
int division = a / b; // Division operator
bool isEqual = (a == b); // Equality operator

C++ Loop Types

C++ supports various types of loops like for, while, and do-while for repetitive tasks. 

// Example C++ loop types
for (int i = 0; i < 5; i++) {
    std::cout << i << " ";
}

int count = 0;
while (count < 3) {
    std::cout << count << " ";
    count++;
}

int num = 0;
do {
    std::cout << num << " ";
    num++;
} while (num < 3);

C++ Decision Making

C++ provides if, else, and switch statements for decision making in your programs. 

// Example C++ decision making
int x = 10;
if (x > 5) {
    std::cout << "x is greater than 5" << std::endl;
} else {
    std::cout << "x is not greater than 5" << std::endl;
}

int choice = 2;
switch (choice) {
    case 1:
        std::cout << "Option 1 selected" << std::endl;
        break;
    case 2:
        std::cout << "Option 2 selected" << std::endl;
        break;
    default:
        std::cout << "Invalid choice" << std::endl;
}

C++ Pointers

Pointers in C++ allow you to work with memory addresses and perform advanced memory manipulation. 

// Example C++ pointers
int value = 42;
int *pointer = &value; // Pointer to int

std::cout << "Value: " << value << std::endl;
std::cout << "Value via pointer: " << *pointer << std::endl;

C++ References

References in C++ provide an alternative way to access the same memory location as another variable. 

// Example C++ references
int number = 10;
int &ref = number; // Reference to int

std::cout << "Number: " << number << std::endl;
std::cout << "Reference: " << ref << std::endl;

C++ Date & Time

C++ offers libraries like <ctime> for working with dates and times. 

// Example C++ date & time
#include 
#include 

int main() {
    std::time_t now = std::time(0);
    std::tm *ltm = std::localtime(&now);

    std::cout << "Year: " << 1900 + ltm->tm_year << std::endl;
    std::cout << "Month: " << 1 + ltm->tm_mon << std::endl;
    std::cout << "Day: " << ltm->tm_mday << std::endl;
    return 0;
}
